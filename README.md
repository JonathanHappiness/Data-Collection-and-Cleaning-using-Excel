# README – Café Sales Data Cleaning Report

*Author:* Jonathan Happiness  
*Excel File:* Link to Spreadsheet  

## Overview
This folder contains the data cleaning report for the Café Sales dataset. The dataset has 10,000 transactional records with information on products, payment methods, and service types. Cleaning ensures the data is accurate and ready for analysis.

## Contents
- *Data Cleaning Report*
  - Cleaning missing Transaction Dates, Items, Payment Methods, Location
  - Standardizing product names and formatting
  - Calculating missing numerical values
  - Removing duplicates
  - Includes a data dictionary

## How to Use
1. Open the Excel file and review cleaned data
2. Use formulas and transformations for reproducibility
3. Use the dataset for analysis and visualization

## Notes / Limitations
- Some missing categorical fields were randomly assigned
- Numerical calculations may introduce minor approximations
- Returning customers cannot be fully tracked
